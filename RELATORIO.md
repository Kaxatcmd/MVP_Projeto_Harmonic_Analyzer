# Relatório de Apresentação do Projeto MVP

## DJ Harmonic Analyzer - Versão Mínima Viável

### Resumo Executivo

O DJ Harmonic Analyzer MVP é uma aplicação de linha de comando desenvolvida em Python que implementa análise automática de características musicais para otimização de mixagem harmônica. O projeto utiliza a biblioteca Librosa para processamento de sinais de áudio, permitindo a extração de duas métricas essenciais: tonalidade musical e frequência de batidas por minuto (BPM).

### Funcionalidades Principais

A solução oferece três funcionalidades nucleares: análise de ficheiros de áudio para determinação da chave musical em notação Camelot, consulta de compatibilidade harmônica entre duas chaves e identificação automática de chaves correlatas. O sistema implementa a Roda de Camelot, sistema amplamente utilizado por profissionais da mixagem digital.

### Arquitetura Técnica

O projeto estrutura-se em módulos especializados: análise audiovisual para detecção de chave e BPM, utilitários para conversão e verificação de compatibilidade harmônica, e interface de linha de comando para interação com utilizador. Todas as dependências encontram-se documentadas em requirements.txt, com Librosa 0.11.0 como dependência primária.

### Conclusão

Este MVP representa uma implementação funcional e minimalista de um sistema de análise harmônica musical, fornecendo fundamentos sólidos para expansão futura com interface gráfica e funcionalidades avançadas de organização e curadoria de conteúdo audiovisual.

**Palavras-chave:** Análise musical, mixagem harmônica, processamento de áudio, Camelot Wheel, Python.
